# Enrai Website

Professional website for Enrai, a DevOps and AI consulting company specializing in training, course content creation, and consulting engagements.

## Website Structure

The website is built using:

- **HTML5**: Structure of the website
- **Bootstrap 5**: CSS framework for responsive design
- **Font Awesome**: Icon library
- **JavaScript**: For interactive elements

## Directory Structure

- `/docs`: The root directory for the website (GitHub Pages ready)
  - `/docs/index.html`: Main HTML file
  - `/docs/css/styles.css`: Custom CSS styles
  - `/docs/js/main.js`: Custom JavaScript functionality
  - `/docs/images/`: Directory for images and media files

## Features

1. **Responsive Design**: Website adapts to different screen sizes
2. **Modern Interface**: Clean, professional appearance using Bootstrap 5
3. **Service Highlights**: Clear sections for DevOps Consulting, AI Integration, and Training offerings
4. **Contact Form**: Form for potential clients to reach out
5. **Interactive Elements**: Smooth scrolling, form validation, and other interactive features

## Getting Started

The website is designed to be hosted directly from the `/docs` folder, making it compatible with GitHub Pages hosting or any standard web hosting service.

## Domain

The website is configured for www.enrai.co
